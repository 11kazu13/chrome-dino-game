# Chrome Dino Game (Canvas)

https://11kazu13.github.io/chrome-dino-game/

Google Chrome のオフライン時に遊べる「Chrome Dino Game」を参考に、  
**HTML / JavaScript / Canvas API** を使って制作したシンプルな2Dゲームです。

Canvas を用いたゲームループ、画像描画、ジャンプ処理など、  
Webアプリ開発の基礎となる要素を一通り実装しています。

---

## 🎮 ゲーム概要

- 恐竜を操作して障害物を避けるシンプルなアクションゲーム
- スペースキーでジャンプ
- Canvas を用いた毎フレーム描画によるアニメーション

---

## 🕹 操作方法

| キー | 動作 |
|----|----|
| Space | ジャンプ |

---

## 🛠 使用技術

- HTML5
- JavaScript (ES6)
- Canvas API
- Git / GitHub
